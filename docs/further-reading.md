# Further reading

> Books, primary papers, and online documentation worth your time. Each entry has a DOI / ISBN / URL so you can find the original.

This is a curated list, not exhaustive. The [Landmark work → Foundational papers](landmark/papers.md) page has a wider literature catalogue.

## Books — neuroimaging

- **Huettel SA, Song AW, McCarthy G.** *Functional Magnetic Resonance Imaging.* 3rd ed. Sinauer Associates; 2014. ISBN 978-0878936274. — the textbook for fMRI physics + analysis.
- **Jones DK.** *Diffusion MRI: Theory, Methods, and Applications.* Oxford University Press; 2010. ISBN 978-0195369779.
- **Poldrack RA, Mumford JA, Nichols TE.** *Handbook of Functional MRI Data Analysis.* Cambridge University Press; 2011. ISBN 978-0521517669.
- **Buxton RB.** *Introduction to Functional Magnetic Resonance Imaging: Principles and Techniques.* 2nd ed. Cambridge University Press; 2009. ISBN 978-0521899956.
- **Friston K, Ashburner J, Kiebel S, Nichols T, Penny W (eds).** *Statistical Parametric Mapping: The Analysis of Functional Brain Images.* Academic Press; 2007. ISBN 978-0123725608.

## Books — data engineering

- **Kleppmann M.** *Designing Data-Intensive Applications.* O'Reilly Media; 2017. ISBN 978-1449373320. — the single most useful book in this list for anyone moving into data infrastructure.
- **Reis J, Housley M.** *Fundamentals of Data Engineering.* O'Reilly Media; 2022. ISBN 978-1098108304. — maps the modern landscape end-to-end.
- **Petrov A.** *Database Internals: A Deep Dive into How Distributed Data Systems Work.* O'Reilly Media; 2019. ISBN 978-1492040347.

## Books — reliability engineering

- **Beyer B, Jones C, Petoff J, Murphy NR (eds).** *Site Reliability Engineering: How Google Runs Production Systems.* O'Reilly Media; 2016. ISBN 978-1491929124. Free online: <https://sre.google/sre-book/>
- **Beyer B, Murphy NR, Rensin DK, Kawahara K, Thorne S (eds).** *The Site Reliability Workbook.* O'Reilly Media; 2018. Free online: <https://sre.google/workbook/>

## Books — machine learning

- **Murphy KP.** *Probabilistic Machine Learning: An Introduction.* MIT Press; 2022. ISBN 978-0262046824. Free draft online: <https://probml.github.io/pml-book/>
- **Goodfellow I, Bengio Y, Courville A.** *Deep Learning.* MIT Press; 2016. ISBN 978-0262035613. Free online: <https://www.deeplearningbook.org/>
- **Hastie T, Tibshirani R, Friedman J.** *The Elements of Statistical Learning.* 2nd ed. Springer; 2009. ISBN 978-0387848570. Free online: <https://hastie.su.domains/ElemStatLearn/>
