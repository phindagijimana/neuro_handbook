# Landmark work

> The papers, pipelines, datasets, and atlases every neuroimaging researcher should know.

Knowing the *literature* is what separates a competent user from someone who can adapt when the tools change. This section is a curated, opinionated tour.

## Chapters

- **[Foundational papers](papers.md)** — what to read, in what order, to understand how the field thinks today.
- **[Reference datasets](datasets.md)** — HCP, UK Biobank, ABCD, ADNI, OASIS, OpenNeuro: what's in each and how to get access.
- **[Major pipelines](pipelines.md)** — FreeSurfer, FSL, AFNI, ANTs, MRtrix3, fMRIPrep, QSIPrep, FastSurfer, MELD, HippUnfold — what each is for.
- **[BIDS-app workflows](bids-apps.md)** — the standardised CLI shape and how to chain BIDS apps together.
- **[Atlases and templates](atlases.md)** — Desikan-Killiany, Destrieux, Schaefer, HCP-MMP, AAL, Glasser; MNI152, fsaverage, fsLR.

## How this section is curated

Each page lists a small number of works with one-line summaries. It is not exhaustive. The goal is "if you read these, you can read the others". Pull requests welcome to add missing pieces.
